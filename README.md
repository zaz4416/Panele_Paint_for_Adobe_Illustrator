# Drawing supporter in Adobe llustrator by Extend Script.

動作環境
Adobe社のイラストレータで動作します。
source-full.zip　をダウンロードしてください。

動作概要
塗り作業をサポートするためのパネルを表示します。
主な機能は、下記の通り。
・カンバスの回転、全体表示
・塗りブラシ、消しゴムなどのツールの選択

動作の開始手順
レイヤーをクリック、その後に、
メニューのファイル　→ スクリプト　→ その他のスクリプトの順番に選び、
Panele_Paint.jsxを選択してください。

ソースファイルの変更はご自由に。なお、非営利でご使用ください。
